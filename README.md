# Red Hat Enterprise Linux (RHEL 9.3) Core Competency: Enterprise Identity, Access Management & Local Security

## 🎯Project Overview
This project validates administrative competencies in process optimization, local Identity & Access Management (IAM), and advanced POSIX permission constraints within Red Hat Enterprise Linux 9.3. The objective is to secure an environment by terminating rogue processes, provisioning administrative users with granular password aging controls, and implementing collaboration directories utilizing special permissions (SGID, Sticky Bit).

## 📋Objectives and Scenario
* **Resource Optimization**: Isolate and terminate high-CPU overhead tasks.
* **IAM Provisioning**: Establish an enterprise group and user account managed under strict security compliance regulations (Sudoers privilege escalation, forced lifecycle aging).
* **Workspace Hardening:** Deploy a shared administrative directory utilizing inheritance masks (Umask), Set Group ID (SGID) enforcement, and data-deletion barriers (Sticky Bit).
